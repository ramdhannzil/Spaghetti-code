# Spaghetti-code
Pertanyaan :
Apa keuntungan menggunakan Class jika perpustakaan memiliki ratusan buku?
----
Jawaban :
Dengan pakai class, kita nggak perlu bikin banyak variabel dan fungsi buat setiap buku. 
Kalau bukunya sampai ratusan, kita tinggal bikin objek baru dari satu class yang sama. 
Kodenya jadi lebih rapih, gampang dipahami, dan lebih mudah dikembangin kalau nantinya ada penambahan fitur atau jumlah buku.
